# Miracle-Diary

습관과 할 일을 정리하는 일기장을 App으로 구현했습니다.

## 컴퓨터 구성 / 필수 조건 (Prerequisites)
Android Version : at least Android API Level 26 (Oreo)

## 설치 (Installation Process)
<a href="/readmeApk/MiracleDiary.apk">미라클 다이어리 다운로드</a>
